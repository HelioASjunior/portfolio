# Portifolio Hélio Júnior

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript, focado em apresentar projetos, certificados e formas de contato profissional.

## Preview

![Preview do portfólio](./assets/mockups/mockup.png)

## Objetivo

Este projeto reúne:

- trabalhos recentes de desenvolvimento
- galeria de certificados com visualização ampliada
- currículo para download
- canais de contato direto

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Swiper (carrossel)
- Font Awesome
- Devicon

## Funcionalidades

- seção de projetos com cards interativos
- seção de certificados com nome dinâmico no hover
- botão Ampliar para abrir certificado em janela flutuante
- layout responsivo para desktop, tablet e mobile
- fundo em vídeo com variação por breakpoint
- botão flutuante de WhatsApp

## Projetos em destaque

- PDF Xino Convert
- Pentest Simulation Kali
- CRM - Escritório Jurídico
- Civil EngAICalc
- Game Underworld Hero
- Malware Analysis (Estudo)
- Wallpaper Animado Steam "Tema Dev"
- Estudo Fast API
- Portfólio Github

## Certificados

A seção de certificados lê os itens definidos em projects.js e exibe os arquivos da pasta assets/certificados.

Atualmente inclui certificados como:

- Administrando Banco de Dados
- AWS Simulearn - Generative AI Practitioner
- Cibersegurança
- Hacking Ético - Vulnerabilidades e Testes de Invasão
- LGPD
- Engenheiro de Agentes de IA
- Git e Github- Controle de Versão e Colaboração

## Estrutura principal

- index.html: estrutura da aplicação
- style.css: estilos globais e responsividade
- projects.js: dados e renderização dos cards
- assets/mockups: imagens dos projetos
- assets/certificados: imagens dos certificados
- assets/curriculo: arquivos do currículo
- assets/bg: vídeos de fundo
